# Utility frameworks

## Modules
 * frameworks-common - Utility Classes;
 * frameworks-aop - Utility Aspect Classes using AspectJ;
 
 ### Build with Maven:
 
 	mvn clean install

The generated files willbe in `target` folder.	 
 
 ### Build with Gradle:
 
 	gradle -x test build

The generated files willbe in `build` folder.
